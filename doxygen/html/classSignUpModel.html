SignUpModel::show ( )

This method is called by the controller and invokes most of the logic.

Creates a new renderer for the page, then checks the fields. If the fields are not validated it calls the same page again, otherwise it tries to add the user to the database and shows a confirmation page. If there's a database error it prints an error message.
